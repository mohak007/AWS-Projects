                </div>
                <div class="service-card">
                    <h3>AWS Lambda</h3>
                    <p>A serverless, event-driven compute service that lets you run code without provisioning or managing servers. You only pay for the compute time you consume.</p>
                </div>
                <div class="service-card">
                    <h3>Amazon RDS</h3>
                    <p>Relational Database Service (RDS) makes it easy to set up, operate, and scale a relational database in the cloud. It manages the database and operating system for you.</p>
